# HKU Health System NPS Calculator

Online tool for clinical units to calculate Net Promoter Score from patient feedback surveys.

## Usage
1. Enter counts: Promoters (9-10), Passives (7-8), Detractors (0-6)
2. Click "Calculate NPS"
3. Use results for governance reports
